Recursion

What is Recursion?

  • Recursion is a technique for solving problems that involves calling a method that calls itself.
  • Recursion is a powerful technique that can be used to simplify a problem.
  • Recursion is a technique that can be used to solve problems that are difficult to solve using other techniques.

Basic Recursive Functions

Let's look at some basic recursive functions.

Factorial

Factorial of a number is the product of all the integers from 1 to that number.

For example, the factorial of 5 is 5 * 4 * 3 * 2 * 1 = 120.

Factorial of 0 is 1.

Factorial of a negative number is undefined.

Here is a recursive function to calculate the factorial of a number.

public static int factorial(int n) {
            if (n == 0) {
                return 1;
            }
            return n * factorial(n - 1);

            }

Let's look at the execution of this function for n = 5.

                        
                            factorial(5)
                            5 * factorial(4)
                            5 * 4 * factorial(3)
                            5 * 4 * 3 * factorial(2)
                            5 * 4 * 3 * 2 * factorial(1)
                            5 * 4 * 3 * 2 * 1 * factorial(0)
                            5 * 4 * 3 * 2 * 1 * 1
                            120

N to 1

Here is a recursive function to print numbers from n to 1.

public static void printNToOne(int n) {
            if (n == 0) {
                return;
            }
            System.out.println(n);
            printNToOne(n - 1);
        }

Let's look at the execution of this function for n = 5.

                        
                            printNToOne(5)
                            printNToOne(4)
                            printNToOne(3)
                            printNToOne(2)
                            printNToOne(1)
                            printNToOne(0)
                            5
                            4
                            3
                            2
                            1

Binary Equivalent

Here is a recursive function to print the binary equivalent of a number.

public static void printBinary(int n) {
            if (n == 0) {
                return;
            }
            printBinary(n / 2);
            System.out.print(n % 2);
        }

Let's look at the execution of this function for n = 5.

                        
                            printBinary(5) // print 1 because 5 % 2 = 1
                            printBinary(2) // print 0 because 2 % 2 = 0
                            printBinary(1) // print 1 because 1 % 2 = 1
                            printBinary(0) // return because n == 0
                            1
                            0
                            1

Recursion and Iteration

Note: anything that can be done in recursion can be done in a iterator

Let's look at the factorial function again.

public static int factorial(int n) {
            if (n == 0) {
                return 1;
            }
            return n * factorial(n - 1);

            }

Here is an iterative version of the factorial function.

public static int factorial(int n) {
            int result = 1;
            for (int i = 1; i <= n; i++) {
                result *= i;
            }
            return result;
        }

Here is a recursive function to print numbers from n to 1.

public static void printNToOne(int n) {
            if (n == 0) {
                return;
            }
            System.out.println(n);
            printNToOne(n - 1);
        }

Here is an iterative version of the printNToOne function.

public static void printNToOne(int n) {
            for (int i = n; i >= 1; i--) {
                System.out.println(i);
            }
        }

Here is a recursive function to print the binary equivalent of a number.

public static void printBinary(int n) {
            if (n == 0) {
                return;
            }
            printBinary(n / 2);
            System.out.print(n % 2);
        }

Here is an iterative version of the printBinary function.

public static void printBinary(int n) {
            while (n != 0) {
                System.out.print(n % 2);
                n /= 2;
            }
        }
